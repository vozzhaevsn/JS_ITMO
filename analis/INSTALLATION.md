# Инструкция по установке Behavior Graph Analyzer

## Предысловия

- Google Chrome версию 88 и выше
- Или любой Chromium-базированный браузер (Edge, Brave, Opera)

## Шаги установки

### 1. Стандартная инсталляция

```bash
# Клонировать репозиторий и навигировать
git clone https://github.com/vozzhaevsn/JS_ITMO.git
cd JS_ITMO/analis
```

### 2. Открыть Chrome Extension Manager

```
рада локации: chrome://extensions/
или edge://extensions/ для Microsoft Edge
```

### 3. Включить "Нестандартные расширения" (девелопер моде)

Нажмите тоггл девелопер моде в верхнем правом углу:

![Developer Mode Toggle]()

### 4. Нажмите "Загрузить распакованное расширение"

Кнопка наравни с заголовком. Нажимают ону.

### 5. Выберите папку `analis`

Навигируйте к папке де вы клонировали репозиторий и выберите папку `analis`.

### 6. Расширение принято!

Вы должны видеть:

```
Behavior Graph Analyzer
Анализ графов поведения пользователей в реальном времени с детекцией ботов
```

## Открытие расширения

1. Нажмите на иконку расширения в панели инструментов Chrome
2. Открывается Popup окно с графом
3. Чтобы сборать данные, навигируйте по веб-сайту

## Троублшутинг

### Проблема: Не почему-то расширение не загружается

**Решение:**
1. Проверьте файл `manifest.json` на синтаксис
2. Откройте девелопер консоль (F12) и просмотрите ошибки
3. Обновите паге в chrome://extensions

### Проблема: Граф не так бостр открываются

**Решение:**
1. В инстраментах разработчика просмотрите Выне о Ошибках
2. Уничтожите один экземпляр в localstorage: актуализируете chrome://extensions

## Мастер-класс для разработчиков

### Дебагинг

Откройте `chrome://extensions` и кликните "Посмотреть в рендере сервиса" для просмотра логов Service Worker.

### Если вы модифицируете код

1. Обновите файлы в папке `analis`
2. В chrome://extensions нажмите кнопку "Обновить" (а иконка с стрелкой)
3. Откройте popup добовою

## Требования к версиям

- **Manifest V3** - новые расширения Chrome
- **Content Scripts** - датэколлекция данных
- **Service Worker** - транспортировка данных
- **IndexedDB** - хранение графов
